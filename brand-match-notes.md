# Ojala Website Branding Notes

The live Ojala website presents a minimalist, high-end beach-neutral experience with large photographic hero imagery, restrained navigation, abundant whitespace, and soft warm neutrals. The page relies on a clean sans-serif system stack in computed styles, with light font weights, generous letter spacing in navigation, and a calm editorial hierarchy.

## Key cues to carry into the operations app

| Element | Observation | Implication for ops app |
| --- | --- | --- |
| Hero treatment | Full-bleed photography with very restrained overlays | The operations entry should feel like a branded subpage, not a separate utility app |
| Typography | Computed body and heading fonts currently resolve to a clean sans-serif system stack with light weights | Update the dashboard and portal to use the same sans-serif-led typography rather than the current serif-heavy editorial treatment |
| Navigation | Sparse, uppercase, letter-spaced navigation with simple buttons | Add a subtle branded access point for login and operations instead of a heavy admin-style header |
| Palette | Warm sand, white, and soft neutral tones | Keep customer-facing and backend surfaces within the same neutral beach palette |
| Brand feel | Contemporary minimalist, refined, airy, approachable | Reduce the visual separation between public site and internal tools so both feel like one Ojala product |
